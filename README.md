# Task Manager API

A secure Task Management REST API built with Spring Boot.

## Features

- JWT Authentication
- Spring Security
- BCrypt Password Encoding
- CustomUserDetailsService
- MySQL Database
- DTOs
- MapStruct
- Swagger/OpenAPI
- Global Exception Handling
- Ownership-based Authorization
- Validation
- Logging

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- JWT (jjwt)
- MapStruct
- Swagger/OpenAPI
- Gradle

Register → Login → JWT Token → Access Protected APIs
